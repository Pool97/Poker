# Poker

Poker è un progetto nato per il corso di **Programmazione Avanzata Java e C** dell'*Università degli Studi di Brescia*.

## Features
* Supporto alla variante Texas Hold'Em
* Supporto alle modalità Fixed Limit e No Limit
* Interfaccia grafica
